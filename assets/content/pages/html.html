O HTML, ou HyperText Markup Language, é a linguagem usada para criar páginas da web. Funciona fornecendo uma estrutura básica para o conteúdo, como texto, imagens e links, permitindo que os navegadores saibam como exibi-los. A diferença entre HTML e HTML5 está principalmente nas funcionalidades adicionais e na capacidade de lidar com conteúdo mais avançado.
